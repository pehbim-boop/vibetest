# 개인 홍보 사이트 제작 명세서

이 문서는 바이브코딩 도구에 그대로 넣어 개인 홍보용 웹사이트를 만들기 위한 명세서다.
결과물은 초보자도 수정하기 쉬운 정적 사이트 형태로 만들어준다.

## 프로젝트 목적

- 나를 처음 보는 사람에게 좋은 첫인상을 주는 개인 홍보 사이트를 만든다.
- 메뉴는 `홈`, `자기소개`, `포트폴리오`, `취미` 4개로 구성한다.
- 전체 분위기는 깔끔하고 부드러운 연녹색 계열로 한다.
- 너무 딱딱한 이력서 느낌보다 친근하고 세련된 자기소개 페이지 느낌으로 만든다.
- 모바일과 PC에서 모두 자연스럽게 보이게 반응형으로 제작한다.

## 제작 방식

- 별도 백엔드 없이 동작하는 정적 웹사이트로 만든다.
- 기본 결과물은 `index.html`, `style.css`, `script.js` 형태를 우선으로 한다.
- 한국어 사이트로 제작한다.
- 코드를 모르는 사람도 텍스트만 쉽게 바꿀 수 있게 구조를 단순하게 유지한다.

## 전체 디자인 방향

- 배경색은 살짝 녹색이 도는 부드러운 톤으로 한다.
- 예시 배경 팔레트
  - 메인 배경: `#f1f7f1`
  - 카드 배경: `#ffffff`
  - 포인트 컬러: `#7aa37a`
  - 진한 텍스트: `#2f4530`
  - 보조 텍스트: `#5f7360`
- 전체적으로 깨끗하고 안정감 있는 느낌을 준다.
- 둥근 카드, 은은한 그림자, 적당한 여백을 사용한다.
- 헤더는 상단 고정 메뉴로 만들고, 클릭 시 각 섹션으로 부드럽게 이동한다.
- 과한 애니메이션은 피하고, 스크롤 등장 애니메이션과 카드 호버 효과 정도만 넣는다.

## 폰트와 분위기

- 제목은 부드럽고 인상적인 느낌, 본문은 가독성 좋은 한글 폰트를 사용한다.
- 예시
  - 제목: `Gowun Batang` 또는 비슷한 분위기의 폰트
  - 본문: `SUIT` 또는 `Pretendard`
- 전체 무드는 `정돈됨`, `친근함`, `신뢰감`, `자연스러움`이다.

## 사이트 구조

한 페이지 스크롤형 랜딩 사이트로 만들고, 상단 메뉴로 각 섹션에 이동한다.

### 1. 홈

홈 섹션에는 첫인상을 만드는 핵심 내용이 들어간다.

- 큰 메인 문구
  - `안녕하세요, 저는 [이름]입니다.`
- 서브 문구
  - `저는 [관심 분야 또는 직무]에 관심이 많고, 사람들에게 좋은 경험을 전하는 일을 좋아합니다.`
- 짧은 소개 문장
  - `차분하게 성장하고, 꾸준히 기록하며, 제 색깔이 담긴 결과물을 만드는 사람입니다.`
- 버튼 2개
  - `자기소개 보기`
  - `포트폴리오 보기`
- 오른쪽 또는 아래에는 심플한 프로필 이미지 영역 또는 일러스트 영역을 둔다.
- 프로필 이미지가 없을 경우에도 어색하지 않도록 원형 그래픽 또는 이니셜 디자인으로 대체 가능하게 한다.

### 2. 자기소개

자기소개 섹션은 너무 길지 않게, 읽기 편한 카드형 또는 2단 구성으로 만든다.

- 섹션 제목: `자기소개`
- 소개 문단 예시
  - `저는 새로운 것을 배우는 과정을 즐기고, 맡은 일을 끝까지 책임감 있게 해내는 사람입니다.`
  - `복잡한 내용을 이해하기 쉽게 정리하고, 작은 아이디어를 실제 결과물로 만드는 일을 좋아합니다.`
  - `함께 일하는 사람에게는 편안하고 믿을 수 있는 사람으로 기억되고 싶습니다.`
- 추가 정보 카드 3개
  - `강점`: 성실함, 빠른 적응력, 꾸준함
  - `관심사`: 콘텐츠, 기획, 협업, 성장
  - `일하는 방식`: 차분하게 정리하고 끝까지 완성하기

### 3. 포트폴리오

포트폴리오 섹션에는 카드 3개를 넣는다.
각 카드는 제목, 한 줄 설명, 역할 또는 포인트, 결과 요약으로 구성한다.

- 섹션 제목: `포트폴리오`
- 카드 1
  - 제목: `[포트폴리오 1 제목]`
  - 설명: `이 프로젝트는 어떤 목적을 가지고 진행했는지 간단히 소개합니다.`
  - 포인트: `기획`, `정리`, `완성도`
  - 결과: `사용자에게 깔끔하고 직관적인 경험을 전달했습니다.`
- 카드 2
  - 제목: `[포트폴리오 2 제목]`
  - 설명: `문제를 해결하거나 정보를 보기 쉽게 구성한 작업을 소개합니다.`
  - 포인트: `문제 해결`, `구조화`, `소통`
  - 결과: `복잡한 내용을 더 쉽게 이해할 수 있도록 만들었습니다.`
- 카드 3
  - 제목: `[포트폴리오 3 제목]`
  - 설명: `작은 아이디어를 실제 결과물로 발전시킨 경험을 소개합니다.`
  - 포인트: `아이디어`, `실행`, `개선`
  - 결과: `실용적이면서도 인상적인 결과물을 만들었습니다.`

추가 요청

- 각 포트폴리오 카드는 이미지가 없어도 완성도 있게 보이도록 만든다.
- 필요하면 썸네일 자리에는 연녹색 계열의 추상 그래픽이나 아이콘을 넣는다.
- 나중에 링크를 추가할 수 있도록 `자세히 보기` 버튼 자리도 남겨둔다.

### 4. 취미

취미 섹션에는 카드 3개를 넣고, 사람적인 매력이 느껴지게 구성한다.

- 섹션 제목: `취미`
- 취미 1: `산책`
  - 설명: `걷는 시간을 통해 생각을 정리하고 새로운 에너지를 얻습니다.`
- 취미 2: `독서`
  - 설명: `다양한 이야기와 관점을 접하며 아이디어를 넓히는 것을 좋아합니다.`
- 취미 3: `사진 기록`
  - 설명: `일상 속 작은 순간을 기록하며 감각과 관찰력을 키웁니다.`

추가 요청

- 취미 카드는 포트폴리오보다 조금 더 부드럽고 감성적인 분위기로 만든다.
- 작은 아이콘이나 일러스트를 함께 사용해도 좋다.

## 레이아웃 가이드

- 상단 고정 네비게이션
- 첫 화면은 화면 높이를 넉넉히 쓰는 히어로 섹션
- 각 섹션은 너무 붙지 않게 충분한 여백 사용
- 포트폴리오와 취미는 카드형 3열 구조
- 모바일에서는 1열 또는 2열로 자연스럽게 변경
- 전체 폭은 너무 넓지 않게 제한해서 읽기 편하게 구성

## 인터랙션 가이드

- 메뉴 클릭 시 해당 섹션으로 부드럽게 스크롤 이동
- 스크롤 시 각 섹션이 자연스럽게 나타나는 효과
- 버튼과 카드에는 가벼운 호버 애니메이션
- 사용자 경험을 해치지 않도록 빠르고 단정한 움직임만 사용

## 문구 스타일

- 말투는 따뜻하고 자신감 있으면서도 과장되지 않게 한다.
- 자기 자랑처럼 보이기보다 신뢰가 가는 톤으로 작성한다.
- 너무 기업형 소개문보다 개인 브랜딩 느낌이 나게 한다.
- 문장은 짧고 읽기 쉽게 정리한다.

## 꼭 반영할 사항

- 메뉴는 반드시 `홈`, `자기소개`, `포트폴리오`, `취미`로 구성한다.
- 배경은 흰색 위주가 아니라 살짝 녹색이 감도는 밝은 톤을 사용한다.
- 포트폴리오는 정확히 3개 보여준다.
- 취미도 정확히 3개 보여준다.
- 초보자도 텍스트만 바꿔서 사용할 수 있게 단순하고 깔끔하게 만든다.
- 결과물은 세련되지만 과하게 화려하지 않게 한다.

## 최종 결과물 기준

- 첫 화면만 봐도 어떤 사람을 소개하는 사이트인지 바로 알 수 있어야 한다.
- 메뉴 이동이 자연스럽고 구조가 직관적이어야 한다.
- 내용이 많지 않아도 허전해 보이지 않아야 한다.
- 색감, 카드, 타이포그래피가 조화롭고 통일감 있어야 한다.
- 실제 배포 가능한 수준의 완성된 프론트엔드 결과물을 만든다.

## 한 줄 요청문

위 내용을 바탕으로 연녹색 감성의 개인 홍보용 반응형 웹사이트를 만들어줘. 메뉴는 홈, 자기소개, 포트폴리오, 취미로 구성하고, 포트폴리오 3개와 취미 3개를 카드형으로 예쁘게 보여줘. 초보자도 나중에 텍스트만 쉽게 바꿀 수 있도록 단순한 구조로 만들어줘.
